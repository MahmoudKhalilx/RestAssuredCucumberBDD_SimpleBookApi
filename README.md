# SimpleBookApi - RestAssured & Cucumber BDD

Test automation framework for the Simple Book API using RestAssured and Cucumber BDD.

## Features
- BDD-style test scenarios using Cucumber
- REST API testing with RestAssured
- Easy-to-read Gherkin syntax
- Reusable step definitions
